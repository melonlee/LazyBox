# LazyBox AI 产品需求文档 (PRD)

## 1. 产品愿景

**将 LazyBox 从 Markdown 编辑器升级为 AI 原生的智能内容创作平台**

### 1.1 产品定位
- **当前定位**: 微信公众号 Markdown 编辑器
- **目标定位**: AI 驱动的全栈内容创作平台，支持多平台发布、智能写作、AI 辅助排版

### 1.2 核心价值主张
> 让创作者专注于思想本身，AI 处理一切繁琐工作

---

## 2. 目标用户

| 用户类型 | 描述 | 核心痛点 |
|---------|------|---------|
| **内容创作者** | 公众号作者、技术博主、专栏作家 | 写作卡壳、排版耗时、灵感枯竭 |
| **知识工作者** | 产品经理、分析师、研究员 | 文档整理困难、知识沉淀零散 |
| **新媒体运营** | 多平台内容运营者 | 跨平台排版繁琐、内容重复创作 |
| **技术文档作者** | 开源项目维护者、API 文档编写者 | 文档更新滞后、版本管理混乱 |

## 3. 核心功能模块

### 3.1 AI 写作助手 (AI Writer)

#### 3.1.1 智能续写
- **功能描述**: 根据当前上下文智能续写内容
- **触发方式**:
  - 快捷键 `Cmd+Shift+A`
  - 工具栏按钮
  - 选中文字后右键菜单
- **能力**:
  - 保持文章风格一致性
  - 支持续写段落、章节
  - 可选择续写长度（短/中/长）

#### 3.1.2 内容润色
- **功能描述**: 优化文字表达，提升可读性
- **能力**:
  - 语法纠错
  - 表达优化（更专业/更通俗）
  - 语气调整（正式/轻松/幽默）
  - 中英互译优化

#### 3.1.3 内容扩写/缩写
- **扩写**: 根据要点生成完整段落
- **缩写**: 提取核心观点，生成摘要
- **风格转换**: 将技术文档转换为博客/教程/社交媒体文案

#### 3.1.4 AI 选题与大纲
- 根据关键词生成文章大纲
- 推荐热门选题方向
- 生成章节结构和要点

#### 3.1.5 引导式写作
- 提供写作提示和问题引导
- 逐步完善文章结构
- 检查逻辑漏洞

### 3.2 AI 生图 (AI Image)

#### 3.2.1 文生图
- 输入描述生成配图
- 支持风格选择（写实/插画/图标/漫画）
- 支持尺寸预设（封面/正文/横版/竖版）

#### 3.2.2 图片优化
- 图片风格转换
- 背景移除/替换
- 图片扩展/补全

#### 3.2.3 图标生成
- 根据概念生成图标
- 批量生成系列图标
- 支持多种导出格式

#### 3.2.4 智能配图建议
- 分析文章内容自动推荐配图位置
- 根据段落内容生成相关插图

### 3.3 AI 排版 (AI Layout)

#### 3.3.1 智能排版
- 一键应用专业排版规则
- 自动处理标题层级、段落间距
- 智能断行和分页

#### 3.3.2 风格模板
- 内置多种排版风格（学术/科技/文艺/商务）
- 用户自定义模板
- 模板市场（分享/购买）

#### 3.3.3 视觉优化
- 自动选择合适的配图位置
- 智能处理引用块、列表样式
- 优化代码块展示

### 3.4 多平台发布 (Multi-Platform)

#### 3.4.1 支持平台
- 微信公众号
- 知乎
- 掘金
- 语雀
- 飞书文档
- Notion
- 个人博客（WordPress/Hugo/Hexo）
- PDF 导出

#### 3.4.2 平台适配
- 自动转换平台特定的格式要求
- 保持内容在不同平台的一致性
- 支持批量发布到多个平台

#### 3.4.3 发布管理
- 定时发布
- 发布历史记录
- 内容版本对比

### 3.5 AI 文件管理 (AI File Management)

#### 3.5.1 智能分类
- 自动分析内容主题
- 智能推荐文件夹位置
- 自动打标签

#### 3.5.2 文档关联
- 发现相关文档
- 构建知识图谱
- 自动生成目录索引

#### 3.5.3 智能搜索
- 语义搜索（不仅匹配关键词）
- 跨文件内容搜索
- 搜索结果高亮和预览

#### 3.5.4 文档整理
- 自动识别重复内容
- 合并相似文档
- 生成文档摘要
- 智能归档旧文档

### 3.6 MCP 插件系统 (MCP Plugins)

#### 3.6.1 MCP 服务器管理
- 添加/删除 MCP 服务器
- 配置服务器参数
- 查看服务器状态

#### 3.6.2 MCP 功能调用
- 浏览可用的 MCP 工具
- 一键调用 MCP 功能
- 查看调用历史

#### 3.6.3 MCP 市场
- 浏览社区 MCP 服务器
- 一键安装配置
- 评分和评论

### 3.7 Claude SKILL 系统 (Skills)

#### 3.7.1 SKILL 管理
- 注册自定义 SKILL
- 导入/导出 SKILL 配置
- SKILL 启用/禁用

#### 3.7.2 内置 SKILL
- 技术文档写作
- 教程编写
- API 文档生成
- 博客文章优化
- 学术论文辅助

#### 3.7.3 SKILL 链式调用
- 组合多个 SKILL 完成复杂任务
- 自定义 SKILL 工作流
- 工作流模板保存

### 3.8 知识库 (Knowledge Base)

#### 3.8.1 向量存储
- 自动索引所有文档
- 向量语义检索
- 跨文档关联发现

#### 3.8.2 知识问答
- 基于个人知识库的 AI 问答
- 快速查找相关信息
- 自动引用来源

#### 3.8.3 知识图谱
- 可视化文档关系
- 追踪思路演变
- 发现隐藏关联

### 3.9 协作功能 (Collaboration)

#### 3.9.1 评论与批注
- 段落级评论
- AI 建议批注
- 评论@提及

#### 3.9.2 版本控制
- Git 集成
- 版本对比
- 一键回滚

#### 3.9.3 团队协作
- 工作区共享
- 权限管理
- 实时协作编辑

## 4. 用户场景

### 场景 1: 从零开始写一篇技术博客
```
1. 用户输入主题 "React 性能优化"
2. AI 生成文章大纲
3. 用户确认大纲，AI 逐节生成内容
4. 用户修改不满意的部分，AI 提供润色建议
5. AI 自动生成配图和代码示例
6. 一键应用技术博客排版模板
7. AI 检查 SEO 优化建议
8. 同时发布到个人博客、掘金、知乎
```

### 场景 2: 整理项目文档
```
1. 用户导入项目文件夹
2. AI 自动分析文档结构，识别重复内容
3. AI 建议文档合并和重组方案
4. AI 自动生成 API 文档目录
5. 用户更新代码，AI 检测到代码变化，提示更新文档
6. AI 为新功能自动生成文档草稿
```

### 场景 3: 内容创作者日常工作流
```
1. 早上 AI 推荐今日热门选题
2. 选择选题，AI 生成初稿
3. AI 优化标题和摘要
4. AI 生成封面图
5. 应用微信公众号排版
6. AI 检查敏感词和违规内容
7. 定时发布到公众号
8. 同步发布到其他平台
```

## 7. 竞品分析

| 产品 | 优势 | 我们的机会 |
|------|------|-----------|
| Notion | 强大的数据库和协作 | 更好的中文支持、AI 原生、微信发布 |
| 飞书文档 | 深度集成飞书生态 | 开放、多平台发布、轻量化 |

| ChatGPT | 强大的 AI 能力 | 专注内容创作、本地文件管理 |

## 8. 风险与挑战

### 技术风险
- AI 生成质量不稳定
- 多平台格式兼容性复杂
- 本地存储与云端同步

### 产品风险
- 用户对 AI 生成内容接受度
- 与现有工具迁移成本
- 功能堆砌导致体验复杂

### 商业风险
- AI 成本高昂
- 用户付费意愿
- 竞品快速跟进
